# 墨香书阁小说阅读平台

免费的在线小说阅读平台，提供海量小说资源的浏览、搜索、收藏和在线阅读服务。采用前端Vue3 + 后端Django的现代化技术架构，支持多种阅读主题和个性化设置。

> **版本**：v1.0 &nbsp;|&nbsp; **技术栈**：Vue3 + Django + MySQL

## 功能模块

### 核心功能

| 模块 | 说明 |
|------|------|
| 首页 | 热门推荐小说展示、分类导航入口、排行榜入口、统计数据区 |
| 书库 | 分类筛选、多维度排序（时间/人气/字数）、分页加载 |
| 小说详情 | 小说信息展示、章节列表、收藏功能、已读/未读区分、书评/评分（1-5星）、平均评分聚合 |
| 阅读器 | 章节阅读、翻页、进度保存、字体/背景/夜间模式、书签管理 |
| 搜索 | 关键词搜索、搜索历史、热门搜索、多维度筛选排序 |
| 排行榜 | 人气榜/新书榜/完结榜/畅销榜四种模式 |
| 用户中心 | 注册登录、个人信息编辑、我的书架、阅读记录、VIP标识 |
| 作者中心 | 我的作品列表、作品/章节 CRUD、封面本地上传、草稿与发布切换、数据看板（作品/章节/字数/阅读量） |
| 评论中心 | 书评发表与展示、1-5星评分、平均分实时计算、删除自己的评论 |

### 阅读器特性

- **字体大小**：4档字号调节（14px/16px/18px/20px）
- **背景主题**：5种阅读背景（默认/护眼/羊皮纸/夜间/深蓝）
- **夜间模式**：深色主题阅读体验
- **进度保存**：自动保存阅读位置，下次打开继续阅读
- **书签管理**：添加/删除书签，快速跳转
- **面包屑导航**：清晰的阅读路径定位

## 快速开始

```bash
# 安装后端依赖
cd novel_backend
pip install -r requirements.txt

# 执行数据库迁移
python manage.py migrate

# 填充测试数据
python manage.py shell < seed_data.py

# 启动后端 (端口8000)
python manage.py runserver 8000

# 安装前端依赖
cd ../novel_frontend
npm install

# 启动前端 (端口5173)
npm run dev
```

打开 `http://localhost:5173` 即可使用。

## API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/novels/` | GET | 小说列表（分页、分类、排序） |
| `/api/novels/{id}/` | GET | 小说详情 |
| `/api/novels/{id}/chapters/` | GET | 小说章节列表 |
| `/api/novels/search/` | GET | 小说搜索 |
| `/api/novels/recommend/` | GET | 推荐小说 |
| `/api/novels/category_stats/` | GET | 分类统计 |
| `/api/chapters/{id}/` | GET | 章节详情 |
| `/api/auth/register/` | POST | 用户注册 |
| `/api/auth/login/` | POST | 用户登录 |
| `/api/auth/logout/` | POST | 用户退出 |
| `/api/auth/me/` | GET | 获取当前用户信息 |
| `/api/favorites/` | GET/POST | 收藏列表/添加收藏 |
| `/api/favorites/check/` | GET | 检查收藏状态 |
| `/api/reading-progress/` | GET | 获取阅读记录 |
| `/api/reading-progress/update_progress/` | POST | 更新阅读进度 |
| `/api/bookmarks/` | GET/POST | 书签列表/添加书签 |
| `/api/author/novels/` | GET/POST | 作者作品列表/创建作品 |
| `/api/author/novels/{id}/` | GET/PATCH/DELETE | 作品详情/修改/删除 |
| `/api/author/novels/{id}/publish/` | POST | 作品发布 |
| `/api/author/novels/{id}/take_down/` | POST | 作品下架 |
| `/api/author/novels/stats/` | GET | 作者数据看板 |
| `/api/author/chapters/` | GET/POST | 章节列表/创建章节 |
| `/api/author/chapters/{id}/` | GET/PATCH/DELETE | 章节详情/修改/删除 |
| `/api/author/chapters/{id}/publish/` | POST | 章节发布 |
