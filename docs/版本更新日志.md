## v1.1（2026-05-29）— Bug 修复 + 功能增强

### Bug 修复（11 项）

**接口层修复：**
- 修复阅读进度 `TypeError: res.forEach is not a function`（DRF 分页对象非数组）
- 修复阅读进度/书签接口 500 错误（序列化器 read_only+write_only 冲突）
- 修复作者中心/章节管理/作品编辑 4 页面双重 `.data` 解包导致 undefined
- 修复书签添加 400 错误（前端字段名 `novel_id` 应为 `novel`）
- 修复小说简介不显示（序列化器缺少 description 字段）

**UI / 体验修复：**
- Google Fonts 超时 → 替换为国内镜像 fonts.loli.net（8 个组件）
- 评论提交按钮过于透明 → 加深+加粗+阴影增强
- IDE 预览面板 React Error #185（IDE 内部问题，非代码问题）

### 新功能（5 项）

1. **404 专用页面**（NotFound.vue）：浮动书本动画 + 返回首页/搜索入口
2. **评论删除按钮**：仅本人可见，ElMessageBox 确认后删除
3. **修改密码功能**：后端 change_password action + 前端弹窗表单，含旧密码校验和 session 保持
4. **小说标签展示**：详情页 Hero 区灰色胶囊标签 + 首页卡片圆角药丸标签
5. **作者页面响应式适配**：4 个页面新增 768px/480px 断点

### 数据库变更
- Bookmark 模型新增 `position` 字段（迁移 0006），记录阅读位置百分比

### 基础设施
- 新增 `.gitignore`
- Git 打标签 v1.0.0
- 新增 NotFound.vue（第 13 个页面）
- 项目文档增至 18 份

## v1.0（2026-05-29）— 正式发布
