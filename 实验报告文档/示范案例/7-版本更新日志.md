## v1.0（2026-05-21）— 项目初始化

- 10个文史哲AI生成模块：哲学降维/典故穿越/文学比较/历史对话/风格转换/知识图谱/古文翻译/概念溯源/文学创作/综合问答
- React + Vite 5 + TypeScript前端三栏SPA布局
- Express + TypeScript后端，提示词路由系统
- LLM抽象层，支持OpenAI兼容API
- 设置面板：API配置 + 提示词模板在线编辑
- API key页面可配，不硬编码
- 文档体系：prd.md → Agent.md → CLAUDE.md

## v12.0（2026-05-25）— 全文搜索高亮 + 多模态视觉 + 展馆系统 + 多模型配置 + 数据全量同步

- **全文搜索高亮**：匹配关键词用 `<mark>` 黄色高亮渲染，支持多词同时高亮，HTML 转义防 XSS
- **多模型API配置**：多文本模型 + 多视觉模型独立管理，每类支持多个配置（端点/Key/模型名），单选激活
- **多模态视觉支持**：图片上传自动压缩（Canvas 800px + JPEG 0.7），IndexedDB 持久化存储，DashScope 原生 API 适配，OpenAI Vision 兼容格式
- **展馆系统**：发布生成结果到展馆，卡片网格展示，结构化预览，查看详细/查看原图，与历史记录双向联动
- **模块名全量同步**：`syncModuleNames()` 每次加载时按 moduleId 同步历史/展馆中的模块名，`updateCustomModule` 改名即时更新历史+展馆，trim 自动清理
- **仪表盘增强**：新增展馆/知识图谱/自定义模块/多模态会话统计卡片，图片存储异步加载，折线图数据点标注值，24h 热力图标注
- **数据联动**：删除历史会话同步清理展馆，`setTokens()` 记录 Token 用量，输出区显示 Token 消耗
- **UI 改进**：主页顶部三标签（会话总次数/总生成量/Token总消耗），展馆按钮 🎨，知识图谱实体排行榜，标签全局删除 ×
