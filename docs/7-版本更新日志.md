# 版本更新日志

## 墨香书阁小说阅读平台

> 按时间顺序记录 v1.0.0 至 v3.0.0 的功能变更。

---

## v1.0.0（2026-05-26）— 基础版本正式发布

- **项目架构搭建**：Django 4.2 + DRF 后端与 Vue 3 + TypeScript + Vite 前端分离架构，Element Plus 组件库，MySQL 数据库
- **核心数据模型**：User 用户表、Novel 小说表、Chapter 章节表、Comment 评论表、Favorite 收藏表，完整 ORM 关联设计
- **首页图书展示**：轮播推荐模块展示热门小说，分类导航快速定位类型
- **排行榜系统**：人气榜和新书榜双榜单展示，实时排序呈现热门内容
- **分类浏览筛选**：按玄幻、都市、言情等分类浏览小说，支持多条件组合筛选
- **全文搜索功能**：支持按书名和作者名关键词搜索，搜索结果高亮匹配
- **用户认证系统**：基于 JWT 的注册登录认证，Token 存储/刷新/过期处理，前端路由守卫权限控制
- **小说详情页**：书籍信息完整展示（封面/简介/作者/分类/章节列表），支持在线阅读
- **章节阅读器**：翻页阅读和目录跳转功能，沉浸式阅读体验
- **收藏功能**：用户可收藏喜欢的小说，个人中心查看收藏列表
- **评论互动系统**：发布评论、点赞评论、删除评论，书签标记阅读进度
- **阅读历史记录**：自动保存用户阅读轨迹，上次看到哪里一目了然
- **作者中心模块**：AuthorCenter 创作工作台，小说 CRUD 操作、封面上传（el-upload + FileReader）、章节 TXT 批量上传（chardet 自动检测编码）
- **后台管理系统**：AdminLayout 侧边栏布局 + Dashboard 仪表盘（用户数/小说数/评论数统计卡片），包含书籍管理、章节管理、广告管理、公告管理、分类管理、违规记录管理、操作日志管理等全套 CRUD 模块
- **后台高级功能**：书籍多条件组合筛选、分页加载排序、批量选择操作、Excel 导出（openpyxl 集成）

## v1.2.0（2026-05-30）— MySQL 迁移与数据扩充

- **数据库迁移**：从 SQLite 迁移至 MySQL 生产环境，编写迁移脚本确保数据无损迁移
- **爬虫封面抓取**：开发 xbiquge.la 定向爬虫批量抓取小说封面图片，实现真实封面覆盖
- **AI 封面生成**：AI 生成封面方案作为补充机制，确保每本小说均有封面展示
- **小说数量扩展**：平台小说从初始数量扩展至 37 本，共计 1212 个章节
- **数据导入优化**：优化 import_novels.py 批量导入脚本流程，完善种子数据初始化配置
- **本地封面匹配**：实现本地封面文件自动匹配功能，提升封面加载效率

## v2.0.0（2026-06-06）— 全面升级与首页重构

- **首页全面重写**：参照七猫免费小说重写 Home.vue，12 大板块设计（轮播 Banner / 热门推荐 / 新书速递 / 分类导航 / 排行榜入口 / 签约作品 / 精选专题 / 改编推荐 / 完本精品 / 作者推荐 / 本周热议 / 更多发现）

- **小说库大幅扩充**：从 17 本扩展至 81 本小说，涵盖玄幻/都市/言情/历史等全品类
- **100% 封面覆盖**：xbiquge.la 爬虫成功爬取 73 本封面共 317 张高清图片，crisp-edges 渲染优化
- **签到系统上线**：SignIn 模型设计 + 视图层 + 序列化器 + 路由配置 + SignIn.vue 前端页面 + 种子数据初始化
- **VIP 充值系统**：Recharge 模型 + 支付逻辑 + 序列化器 + API 路由 + Recharge.vue 充值页面 + 套餐种子数据
- **个人中心重构**：UserCenter.vue 全面整合用户信息/收藏/历史/设置等功能模块
- **圆形头像上传**：el-upload 结合 FileReader 实现图片裁剪预览上传
- **请求稳定性修复**：Books.vue 和 Chapters.vue 从原生 fetch 统一改用封装的 request.ts，解决异步卡死问题
- **导航栏头像修复**：修复 4 处头像显示不正确的 bug，统一界面一致性
- **后台用户名修复**：修复管理后台右上角用户名硬编码"行政"问题，正确显示当前登录用户
- **个人中心接口修复**：通过 AdminUser 类型守卫解决收藏/阅读历史/书签 500 错误
- **收藏删除修复**：修正 novel_id 字段名错误，解决"记录不存在"异常
- **分页性能优化**：修复仅显示 17 条数据的分页异常，配置 PAGE_SIZE_QUERY_PARAM 参数
- **编辑按钮优化**：书籍表格操作列新增编辑按钮，经 4 次迭代（link→solid→plain→overflow:visible）
- **空值保护加固**：Home.vue 添加 SAFE_DEFAULTS 默认值对象和可选链操作符兜底处理
- **URL 路由修复**：urls.py base_name 参数缺失修复，确保 URL 反向解析正常工作

## v3.0.0（2026-06-07）— 企业级全面升级

### 新增功能
- **关注作者系统**：UserFollow 模型 + follow/unfollow/check 等 6 个 RESTful API 接口，前端关注/取消关注按钮组件，AuthorProfile.vue 作者主页（展示作者信息+作品列表），MyFollows.vue 我的关注页，管理端粉丝列表管理
- **小说评分系统**：NovelRating 模型 + submit/rate/list 3 个 API 接口，星级评分弹窗组件（1-5 星选择），详情页平均分展示与评分人数统计，支持评分修改更新
- **签到会员体系**：每日签到功能（CheckIn 组件替代原 SignIn），会员权益等级体系，后台签到管理页面（Membership 管理）
- **订单管理后台**：OrderManage 管理页面，支持订单查看与管理操作
- **作者主页**：AuthorProfile.vue 独立页面，展示作者头像/简介/作品列表/关注状态
- **我的关注页**：MyFollows.vue 页面，卡片式展示已关注作者列表，支持取消关注操作

### UI/UX 企业级重构
- **首页 Home.vue 重构**：深蓝主题色 #2563EB，Banner 渐变背景效果，新上架横向拖拽滚动卡片（overflow-x:auto），热门榜单 Tab 切换组件，书单推荐区（1 主推大卡片 + 5 网格小卡片布局）
