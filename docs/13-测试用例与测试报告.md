# 测试用例与测试报告

## 墨香书阁 — 企业级高品质小说阅读网站

---

## 一、测试概况

本报告记录墨香书阁 v3.0.0 项目的测试范围、测试方法和测试结果。测试采用人工验证方式，在主流浏览器环境下执行，覆盖项目中确定的全部功能区域。

测试环境为本地开发环境（前端Vite开发服务器 + Django后端API服务），使用Chrome、Edge、Firefox三种浏览器分别验证。测试数据库为MySQL，测试数据包含81本小说、2381章节、12分类的完整数据集。

---

## 三、测试用例设计

### 3.1 用户系统（12项）

**注册与Session登录测试（4项）**：分别测试正常注册（用户名+密码+确认密码）、重复用户名注册、密码格式校验、注册后自动登录流程。使用POST /api/auth/register/接口提交表单数据，验证点：（1）注册成功后返回用户信息并建立Session会话；（2）重复注册时提示"用户名已存在"；（3）密码长度不足时前端即时校验拦截；（4）注册成功后自动跳转首页并保持登录状态。

**登录认证测试（3项）**：测试正确凭据登录、错误密码登录、未注册账号登录。使用POST /api/auth/login/接口，验证点：（1）登录成功返回用户信息并建立Session；（2）密码错误提示"用户名或密码错误"；（3）request.ts的401响应拦截器正确触发跳转到登录页。

**个人中心测试（3项）**：进入个人中心页面，验证点：（1）用户信息展示（用户名、邮箱、注册时间）完整准确；（2）个人信息编辑（PUT /api/auth/update_profile/）正常工作；（3）收藏列表和阅读历史展示正确。

**头像上传测试（2项）**：使用el-upload组件进行头像上传测试。验证点：（1）beforeAvatarUpload正确校验文件类型（仅允许jpg/png）和文件大小（≤2MB）；（2）FileReader API实现本地预览，选择图片后立即显示预览图。

### 3.2 阅读功能（18项）

**深蓝首页测试（4项）**：打开首页，验证点：（1）深蓝主题样式正确应用，整体视觉效果专业沉稳；（2）轮播推荐组件自动播放且切换流畅；（3）人气榜单和新书榜单数据加载完成且排序正确；（4）书单推荐模块展示正常。

**金色书库测试（5项）**：进入书库页面，验证点：（1）金色主题样式正确应用，视觉效果高端大气；（2）分类筛选（12分类）正常工作，只显示选中分类的小说；（3）横向拖拽卡片功能正常，可流畅左右拖动浏览小说卡片；（4）排序功能（最新/最热/完结）正常工作；（5）分页浏览功能正常。

**金色详情页测试（4项）**：从书库点击小说卡片进入详情页。验证点：（1）金色主题样式正确应用；（2）封面图、书名、作者、简介、字数等信息完整展示；（3）章节列表按顺序排列并可滚动浏览；（4）关注作者按钮和评分按钮显示正常。

**章节阅读器测试（3项）**：进入章节阅读器页面。验证点：（1）正文内容排版整齐、字体清晰可读；（2）点击"上一章"/"下一章"按钮翻页响应时间≤0.5秒；（3）阅读进度自动保存，再次打开时定位到上次阅读位置。

**搜索与榜单测试（2项）**：验证点：（1）关键词搜索响应≤1秒，结果高亮显示；（2）排行榜Tab切换（人气榜/新书榜/完结榜/畅销榜）数据动态刷新无卡顿。

### 3.3 社交互动（16项）

**关注作者系统测试（6项）**：基于6个关注作者API的完整流程测试。验证点：（1）GET /api/authors/ 获取作者列表正常；（2）POST /api/follow/ 关注作者成功，按钮状态变为"已关注"；（3）DELETE /api/follow/{id}/ 取消关注成功，按钮状态恢复；（4）GET /api/follow/ 我的关注列表展示正确；（5）GET /api/authors/{id}/ 作者主页展示粉丝数和作品列表；（6）关注状态在页面刷新后保持。

**评分系统测试（3项）**：基于3个评分API的功能测试。验证点：（1）星级评分弹窗打开正常，可选择1-5星评分；（2）POST /api/ratings/ 提交评分成功，评分数据显示在详情页；（3）GET /api/novels/{id}/ratings/ 获取评分列表正常，平均分计算正确。

**评论系统测试（4项）**：在小说详情页评论区。验证点：（1）登录用户可发布评论，内容长度校验正常；（2）评论发布后即时显示在评论列表中；（3）每条评论显示用户头像、用户名、内容和发布时间；（4）评论者可删除自己的评论。

**作者主页测试（3项）**：进入作者主页。验证点：（1）作者基本信息（笔名、简介）展示完整；（2）作品列表按时间倒序排列；（3）关注按钮和粉丝数显示正确。

### 3.4 用户运营（10项）

**每日签到测试（5项）**：进入签到页面。验证点：（1）签到按钮点击后状态变为"已签到"，显示连续签到天数；（2）连续签到奖励递增显示正确；（3）签到历史记录按日期倒序排列；（4）断签后重新计数逻辑正确；（5）签到奖励领取后积分余额更新。

**会员中心测试（3项）**：进入会员中心页面。验证点：（1）会员套餐展示（月卡/季卡/年卡）信息完整；（2）会员权益标识在个人中心和各页面正确显示；（3）会员状态与权限关联正确。

**个人中心整合测试（2项）**：验证点：（1）个人中心集成签到入口和会员入口，导航流畅；（2）我的收藏、阅读历史、设置等功能模块布局合理。

### 3.5 管理后台（20项）

**数据仪表盘测试（3项）**：打开仪表盘页面，验证点：（1）统计卡片显示总用户数、总小说数（81本）、总章节数（2381）、今日新增等核心指标；（2）数据与实际数据库一致；（3）图表可视化渲染正常。

**书籍管理测试（5项）**：管理员登录后台管理系统。验证点：（1）书籍列表支持高级筛选（按分类/状态/作者等条件组合查询）；（2）批量操作（批量审核/批量删除）选中多条记录后一键执行；（3）单本书籍编辑操作正常；（4）分页组件显示总记录数和当前页码；（5）Excel导出功能正常。

**章节管理测试（3项）**：后台章节管理模块。验证点：（1）按小说筛选查看其全部章节；（2）可编辑和删除任意章节内容；（3）章节内容展示正常。

**用户管理测试（3项）**：验证点：（1）用户列表展示所有注册用户；（2）可查看用户详细信息；（3）用户搜索和筛选功能正常。

**分类管理测试（3项）**：验证点：（1）12个小说分类完整显示；（2）新增/编辑/删除分类操作正常；（3）前台分类导航即时刷新。

**管理布局测试（3项）**：验证点：（1）后台导航菜单完整显示所有管理入口；（2）侧边栏路由切换正常；（3）用户下拉菜单显示当前管理员信息。

### 3.6 UI交互（8项）

**横向拖拽卡片测试（2项）**：在书库页面测试横向拖拽功能。验证点：（1）鼠标拖动时卡片跟随移动流畅；（2）释放鼠标后卡片自动对齐到最近位置。

**星级评分弹窗测试（2项）**：在详情页测试评分弹窗。验证点：（1）点击评分按钮弹出星级选择窗口，可选择1-5星；（2）提交评分后弹窗关闭，详情页显示最新评分。

**企业级UI主题测试（2项）**：验证点：（1）深蓝首页色调统一，视觉专业沉稳；（2）金色书库和详情页色调统一，视觉高端大气。

**响应式三端适配测试（2项）**：分别在桌面端（1920x1080）、平板端（768x1024）、移动端（375x667）下测试。验证点：（1）三种设备下页面布局自适应正常；（2）核心功能（浏览/搜索/阅读）在三端均可正常使用。

### 3.7 系统基础功能（8项）

**API封装验证测试（3项）**：验证点：（1）所有API请求baseURL=/api正确配置；（2）withCredentials=true跨域凭证携带正常；（3）请求超时设置合理，401状态码响应拦截器触发路由跳转。

**TypeScript编译测试（2项）**：验证点：（1）执行npm run build或TypeScript类型检查，编译零错误；（2）所有Vue组件的类型定义完整且正确。

**兼容性测试（3项）**：分别在Chrome（120+）、Edge（120+）、Firefox（120+）中打开系统。验证点：（1）核心阅读流程在三款浏览器中均正常运行；（2）Element Plus UI组件在各浏览器中渲染一致；（3）Vue Router路由切换动画流畅无卡顿。

## 五、测试结论

**系统达到验收标准，可通过验收。**
